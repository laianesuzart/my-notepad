## about

My Notepad is a simple and user‚Äëfriendly application that lets you add to-dos (checklists) and notes by pressing Enter. To edit an item, just double-click the card and press Enter to save or Esc to cancel. You can also delete items using the Delete button, which becomes visible when hovering over a card.

For to-do cards, you can mark tasks as completed by checking the checkbox. You may also delete completed tasks ‚Äî or all tasks at once ‚Äî using the actions below the input field.

You can also reorder your to-dos by dragging the reorder icon on each item.

The platform includes light and dark themes, and you won‚Äôt need to switch them every time ‚Äî your theme preference is saved in localStorage. All your to-dos and notes are also stored locally, so feel free to add as much as you want! üòÑ

## getting started

To run the project locally, make sure you have [Node.js](https://nodejs.org/en/) and a package manager such as [npm](https://docs.npmjs.com/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/laianesuzart/my-notepad.git
   ```
2. Navigate to the project folder:
   ```bash
   cd my-notepad
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
